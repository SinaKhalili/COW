# Additions
It adds command line argument support! (But only a single argument)

For example ` ./cow.out "G" ` 
will initialize the memory blocks to `0|71|0|0`<- this is where you begin (71 is the ascii code for G)


This allows the user to supply a single string as an argument to a cow program.


The only motivation for this was to write the classic [cowsay program in cow itself](https://github.com/SinaKhalili/cowsay-but-in-cow).


COW is a toy esoteric language. See the [classic COW webpage](http://bigzaphod.github.com/COW/) for more info!

## Usage

In the `source` directory there is a cowcomp.cpp file, just compile it with
```bash
g++ cowcomp.cpp -o cowcc
```
And you have the cowcc cow compiler! Add it to your PATH to cow compile anywhere or just call with with 
```bash
./cowcc program.cow
```

Just like putting river water in your socks, writing and compiling a cow program is fun, easy, and free!
You just give it a text file!
